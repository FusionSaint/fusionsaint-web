## Local dev
1. Copy `.env.example` to `.env` and fill:
   - `PUBLIC_SANITY_PROJECT_ID`
   - `PUBLIC_SANITY_DATASET` (e.g., `production`)
2. Install & run:
   ```bash
   npm i
   npm run dev
   ```

## Build
```bash
npm run build
npm run preview
```
